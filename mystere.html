-- Jeu mystère --

Cette section de jeu photo mystère est pour vous une manière ludique de découvrir certaines photos.

Pour moi, c'est également une excellente opportunité pour m'exercer à coder un jeu.

Le principe est simple : essayez de deviner ce que représentent les images affichées. Vous avez trois essais. Le premier sans aide, puis les deux suivants avec un indice.

Vous pouvez donner jusqu'à trois réponses, un mot suffit.
